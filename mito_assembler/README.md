# Desc.

This is a tiny "disassembler" of plant mitochondial genome.

# Requirements

- Rust lang
- SMRTPipe version 6.0.
- Fastq file for target organism

- Docker or the softwares below:
  - minimap2
  - LAST
  - Flye
